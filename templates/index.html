Тема - глобальное потепление

Про сайт

Этот сайт даст ответы на часто задаваемые людьми вопросы по теме глобального потепления. Если того, что вы ищете, нет на сайте, задавайте свой вопрос в комментарии.


Вопросы



Что такое глобальное потепление? {% if question1%}
Это повышение средней температуры на Земле, которое фиксируется с конца XIX века. {% endif %}

Неужели совсем небольшое повышение средней температуры на что-то влияет? {% if question2%}
Все изменения климата, которые мы на себе ощущаем, являются последствиями глобального потепления. {% endif %}

Что стало причиной глобального потепления? {% if question3%}
Глобальное потепление спровоцировано бурным развитием промышленности. Чем больше выбросов от предприятий, чем активнее идёт вырубка лесов, тем больше накапливается парниковых газов. И тем сильнее прогревается Земля. {% endif %}

Изменение климата – это глобальное потепление? {% if question4%}
Глобальное потепление – основная причина современных изменений климата. Однако изменение климата – более широкое понятие. Оно включает не только рост температуры приземных слоёв атмосферы, а значит, суши и океана, но и его последствия {% endif %}

Когда началось изменение климата? {% if question5%}
Климат менялся всегда. Однако температура из-за естественных циклов не поднималась так быстро за последние 65 миллионов лет. {% endif %}

Какие страны наиболее страдают от глобального потепления? {% if question6%}
Наиболее уязвимы к глобальному потеплению малые островные государства. Это страны в Тихом и Индийском океанах, Средиземноморье, Южно-Китайском море и Карибском бассейне. {% endif %}

Можно ли при помощи современных достижений науки остановить глобальное потепление? {% if question7%}
Полностью остановить процесс изменений климата, вызванных человеком, уже невозможно, однако сейчас можно приостановить процесс потепления до относительно безопасного уровня {% endif %}

Каким странам сейчас выгодно глобальное потепление? {% if question8%}
Ответ очевиден - никаким. {% endif %}

Было ли изменение климата до появления человека? {% if question9%}
Было, однако оно не так быстро происходило, процесс не был таким глобальным. {% endif %}

К чему всё это может привести? {% if question10%}
Учёные прогнозируют, что дальнейшее глобальное потепление может усилить губительные для людей процессы, спровоцировать засухи, наводнения, молниеносное распространение опасных болезней. {% endif %}
